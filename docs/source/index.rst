SAAL - Sistema de Apoio à Administração Legislativa
===================================================

O SAAL é o sistema do Interlegis para Apoio à Administração Legislativa
e visa informatizar as áreas de gestão de:

* Orçamento
* Contabilidade
* Recursos Humanos
* Folha de Pagamento
* Patrimônio
* Material, e
* Tesouraria das Casas Legislativas.
