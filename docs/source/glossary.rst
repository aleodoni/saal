Glossário
=========

.. glossary::

   breadcrumbs
      É um elemento de navegação estrutural que proporciona um meio de
      localização e movimentação dentro da estrutura do sistema.
      
   dashboard
      É uma tela composta de uma ou mais camadas, sob a forma de um painel,
      com elementos visuais como gráficos que mostram a evolução de variáveis,
      ou tabelas que apresentam resumos.
      
   login
      Derivado do inglês *log in*, sendo por vezes também utilizada a
      alternativa *log on*, e de forma menos comum *sign in*, é o processo
      através do qual o acesso a um sistema é controlado por meio da
      identificação e autenticação do utilizador através de credenciais
      fornecidas por esse mesmo utilizador. Essas credenciais são um Nome de
      Usuário ou apenas Usuário (do inglês *username*) e uma senha (no inglês 
      *Password*).

   menu suspenso
      Um menu suspenso é um elemento de interface com o usuário similar à uma
      lista, que permite que o usuário escolha um valor de uma lista de opções
      que "cai para baixo". Quando o menu suspenso está inativo, ele esconde
      as opções do menu, economizando espaço na tela. Quando ativado, ele
      mostra uma lista de opções que "cai", onde o usuário pode selecionar uma
      das opções para que o sistema possa interpretar a intenção do mesmo.
